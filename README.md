# LuminaBook-LED-Bulb-Booking-Website
LuminaBook – LED Bulb Booking Website
💡 LuminaBook – LED Bulb Booking Website

An interactive LED bulb booking website built using HTML5, CSS3, and Vanilla JavaScript.

✨ Features

- 💡 Interactive LED bulb with ON/OFF animation
- 📋 LED bulb booking form
- 🏷️ 100 dynamically generated LED bulb models
- 💰 Automatic total price calculation
- 📅 Delivery date selection
- 🌙 Dark / Light mode with Local Storage
- 💬 Floating customer support widget
- ✅ Booking confirmation modal
- 📱 Fully responsive design
- 🎨 Modern UI with smooth animations

🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

📂 Project Structure

project/
│── index.html

Everything (HTML, CSS, and JavaScript) is contained in a single file.

🚀 How to Run

1. Clone the repository.

git clone https://github.com/your-username/luminabook.git

2. Open "index.html" in your web browser.

No installation or dependencies are required.
